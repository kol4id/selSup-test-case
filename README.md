### 1. Склонировать репозиторий  
```bash
git clone https://github.com/kol4id/selSup-test-case
cd selSup-test-case
```
### 2. Установить зависимости
```bash
npm install 
```
### 3. Запустить сервер
```bash
npm run dev
```
### 4. Открыть

приложение доступно по адресу:
```bash
http://localhost:5173
```
